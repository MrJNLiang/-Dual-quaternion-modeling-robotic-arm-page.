# 代理模式任务提示词

你是该项目的后续开发代理，请在不破坏现有 UI 结构的前提下继续扩展。目标是保持“公式 ↔ UI ↔ 3D”严格对应。

## 任务目标
- 进一步完善 dual quaternion (qt) 的教学可视化。
- 保留 4-DOF (3R + 1P) 结构、乘法链、误差右乘模型与轨迹对比。
- 每个新增公式都要有对应 UI 展示与可编辑输入。

## 工作要求
1) 不改动已有文件结构：
   - `3d_dual_qt/index.html`
   - `3d_dual_qt/styles.css`
   - `3d_dual_qt/app.js`

2) 如新增功能需要新文件，先说明原因，再创建。

3) 新增功能的优先级建议：
   - 精确 ScLERP 与 twist log/exp
   - 目标轨迹与 H∞ 控制闭环仿真
   - 更详细的误差建模可视化

4) 所有新增交互都要：
   - 在 UI 里可见
   - 在 3D 里有对应几何或轨迹体现
   - 与数学公式可互相代入

5) 保持配色与排版风格一致。

## 输出要求
- 修改后提供简要说明（改了什么、为什么）。
- 若添加新公式，给出 LaTeX 版本与数值展示字段。
